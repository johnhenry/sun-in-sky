## Key Features Summary

### What Students Can Do
- Switch between daily (24 hour) and yearly (365 day) views
- Adjust latitude from South Pole to North Pole
- Change Earth's axial tilt (or simulate other planets)
- Select any date and time of year
- Use preset buttons for key dates (solstices, equinoxes)
- Animate sun movement with play button
- See real-time statistics (altitude, declination, daylight hours)

### What Teachers Can Demonstrate
- Why seasons exist (Earth's tilt + orbit)
- How latitude affects climate
- Midnight sun and polar night phenomena
- Equinoxes vs. solstices
- How other planets would experience seasons
- The relationship between tilt and Arctic/Tropic locations

## Frequently Asked Questions

### About the App

**Q: Do students need to install anything?**
A: No! It's a web app. Just share the URL.

**Q: Does it work on phones/tablets?**
A: Yes! It's fully responsive and works on any device with a modern browser.

**Q: Is it free?**
A: Yes, completely free to use and deploy.

**Q: Can I use it offline?**
A: After the initial load, it works without internet. For reliable offline use, run it locally with `npm run dev`.

### About Teaching

**Q: What grade level is this for?**
A: Primarily grades 6-12, but adaptable for younger or older students.

**Q: How much class time do I need?**
A: Activities range from 5 minutes to 90 minutes. Choose based on your needs.

**Q: What if my students don't understand the graph?**
A: Start with the 24-hour view and use preset buttons. The Quick Reference has simplified explanations.

**Q: Can students explore on their own or should I guide them?**
A: Both! Start with teacher-led demos, then allow guided exploration with specific tasks.

### About Deployment

**Q: Which deployment platform should I use?**
A: GitHub Pages (free, reliable), Netlify (easiest), or Vercel (modern). All are free and good options.

**Q: What if my school blocks the deployment URL?**
A: Contact IT with the educational purpose. Deploy to multiple platforms as backups. Run locally if needed.

**Q: How do I update the site after deploying?**
A: See the "Updating Your Deployed Site" section in the Deployment Guide.
